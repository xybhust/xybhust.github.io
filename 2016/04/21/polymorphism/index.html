Polymorphism quickguide

Brief review of polymorphism mechanism in C++.

In C++, polymorphism refers to the situation when a parent-type pointer points to an inherited object or an inherited object (pointer or reference) is passed where the base object is expected.

Case 1: pointers

In this case, no matter wether the display() in the parent class (account) is virtual or not, the output is “credit”.

However, things will become more complicated in the following case

  1. If display() in the parent class (account) is virtual:
  • If the inherited class has the same member function, then the output is “credit”.
  • If the inherited class doesn’t have the same member function, then the output is “account”.
  1. If display() in the parent class (account) is NOT virtual:
  • No matter whether the inherited class override the member function or not, the output is “account”.
