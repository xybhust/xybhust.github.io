Implementation of logistic regression using Theano

Using stochastic gradient descent to train the logistic regression.

Logistic Function

$$ h_{\theta}(x) = \frac{1}{1 + e^{-\theta^Tx}} $$

Loss Function

$$ L(\theta) = \sum_{i=1}^{m}\left\{y^i \log{h\left(x^i \right)} + \left(1-y^i \right)\log{\left(1-h\left(x^i \right ) \right )} \right \} $$

Stochastic Gradient Descent Update Rule

$$ \theta_j := \theta_j + \alpha\left(y^i - h_{\theta}\left(x^i \right ) \right )x_j^i $$
