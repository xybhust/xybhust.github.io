Algorithm Analysis

This article summaries the basics about algorithm analysis.

An algorithm is a clearly specified set of simple instructions to be followed to solve a problem.

Mathematical Background

Definition 1
$T(N) = O\left(f(N) \right )$ if there are positive constants $c$ and $n_0$ such that $T(N) \leq O\left(f(N) \right )$ when $N \geq n_0$.
Definition 2
$T(N) = \Omega\left(f(N) \right )$ if there are positive constants $c$ and $n_0$ such that $T(N) \geq O\left(f(N) \right )$ when $N \geq n_0$.
Definition 3
$T(N) = \Theta\left(f(N) \right )$ if and only if $T(N) = O\left(f(N) \right )$ and $T(N) = \Omega\left(f(N) \right )$.
Definition 4
$T(N) = o\left(f(N) \right )$ if, for all positive constant $c$, there exists an $n_0$ such that $T(N) < O\left(f(N) \right )$ when $N > n_0$. Less formally, $T(N) = o\left(f(N) \right )$ if $T(N) = O\left(f(N) \right )$ and $T(N) \neq \Theta\left(f(N) \right )$.
